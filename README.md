# Data Modeler - MongoDB Blog Schema

This project demonstrates how to design a MongoDB schema for a blogging platform using Mongoose. The schema defines the structure for storing blog posts including title, content, author, tags, and creation date.

## Features

* MongoDB schema design
* Blog data structure
* Mongoose schema definition
* Organized project structure

## Technologies Used

* Node.js
* MongoDB
* Mongoose

## Project Structure

Data-Modeler-Blog-Schema

│

├── models

│   └── blogModel.js

│

├── server.js

├── package.json

└── README.md

## Schema Fields

title – Blog title
content – Blog post content
author – Blog author
tags – Blog categories or tags
createdAt – Date when the blog was created

## Learning Outcome

This project helped in understanding how MongoDB schemas are designed using Mongoose and how blog data can be structured efficiently for a blogging platform.
